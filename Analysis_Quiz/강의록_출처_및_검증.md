## 🔄 **2. SCU_RSTCON2.B.USRINFO 레지스터 사용**

### ✅ **출처 위치**
- **파일**: `OTA_Bootloader.md` 
- **섹션**: `# Practice : Programming Session (App)`
- **라인**: 약 2033줄

### 📋 **강의록 원문**
```
Programming Session 에 반응하여, 아래와 같이 동작하도록 구현 하세요, 
Programming Session 요청에 대한 응답 SCU RSTCON2 레지스터의 USRINFO Field 에 0x1 을 병고 Reset 수행
```

### 🔍 **세부 코드 (App에서의 동작)**

#### 출처: 라인 약 2090~2125
```c
else if(stDiagStatus.currentSession == DIAG_SESSION_PROGRAMMING) {
    Mcal_ResetSafetyENDINIT_Timed(MCU_SAFETY_ENDINT_TIMEOUT);
    SCU_RSTCON2.B.USRINFO = RESET_REASON_REPROGRAMMING_REQUEST;  // ← 0x1 설정
    Mcal_SetSafetyENDINIT_Timed(); 
    Mcu_PerformReset();
}
```

### 💡 **의미**
Application 코드가 Programming Session을 감지하면:
1. RSTCON2 레지스터의 USRINFO 필드에 0x01 (REPROGRAMMING_REQUEST) 저장
2. MCU 리셋 실행
3. Bootloader가 시작되면서 USRINFO가 0x01인지 확인 → Programming 모드 유지

## 🚀 **3. Application 시작 주소 정의**

### ✅ **출처 위치**
- **파일**: `OTA_Bootloader.md`
- **섹션**: `# Bootloader To Application Code\Ota_Boot_1`
- **라인**: 약 2440줄

### 📋 **강의록 원문**
```
#define MEMORY_ADDRESS_APPLICATION 0x80100000u    Definition.h

void EA_initMcu(void) {
    FP_void_void entrypointApplicaton = (void*) MEMORY_ADDRESS_APPLICATION;
    if(SCU_RSTCON2.B.USRINFO != RESET_REASON_REPROGRAMMING_REQUEST) {
        entrypointApplicaton();   // ← App 시작!
    } else {
        // Reprogramming 모드: Bootloader에서 대기
    }
}
```

### 💡 **의미**
- App의 첫 번째 명령어(Entry Point)는 **0x80100000 주소**에 위치
- Bootloader가 함수 포인터로 이 주소로 점프하여 Application 실행
- REPROGRAMMING_REQUEST가 아니면 무조건 App 실행

## ⏰ **5. Programming Session 타임아웃 (5초)**

### ✅ **출처 위치**
- **파일**: `OTA_Bootloader.md`
- **섹션**: `# Bootloader To Application Code\Ota_Boot_1`
- **라인**: 약 2450줄

### 📋 **강의록 원문**
```c
#define TIME_DIAG_S3_SERVER TIME_5S     // ← S3 타임아웃: 5초

typedef struct {
    uint32 timeS3ServerStart;
} ST_BswDcmData;

void REcbBswDcm_initialize(void) {
    stBswDcmData.stDiagStatus.currentSession = DIAG_SESSION_PROGRAMMING;
}

void REtBswDcm_bswDcmMain(void) {
    if(UT_getTimeEcuAlive1ms() > (stBswDcmData.timeS3ServerStart + TIME_DIAG_S3_SERVER)) {
        stBswDcmData.stDiagStatus.currentSession = DIAG_SESSION_DEFAULT;  // ← 5초 후 자동 복귀
    }
}
```

### 💡 **의미**
- Bootloader가 시작되면 Programming Session 진입
- 5초 이내에 새로운 UDS 명령이 없으면 자동으로 DefaultSession으로 변경
- DefaultSession이 되면 Application 실행

## 🔗 **6. 링커 스크립트의 RESET 주소**

### ✅ **출처 위치**

- **섹션**: `# Linker Script (App) : To-Be`
- **라인**: 약 1934줄

### 📋 **강의록 원문**
```c
/* OTA_APP_3 (Application) */
#define INTTAB    0x8017E000    // 인터럽트 테이블
#define RESET     0x80100000    // ← App 시작 주소
#define IFXTRAPTAB (INTTAB + 0x1800)

/* OTA_BOOT_0/1 (Bootloader) */
#define INTTAB    0x80026000    // 인터럽트 테이블
#define RESET     0x80000020    // ← Boot 시작 주소
#define IFXTRAPTAB (INTTAB + 0x1800)
```

### 💡 **의미**
- **RESET은 MCU가 전원 켜질 때 첫 실행 주소**
- Bootloader: 0x80000020 → Bootloader 실행
- Application: 0x80100000 → Application 실행 (Bootloader가 점프)
